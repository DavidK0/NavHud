Current features:
 - equatorial grid and local horizontal grid (switchable) wireframe spheres
 - attitude indicator
 - togglable zoom-based or fixed size for the grids and attitude indicator.
 - Prograde/retrograde, normal/antinormal, radial in/out.
 - Option to show only some indicators, e.g. only prograde/retrograde.
 - Option to disable grid lines and show only attitude markers.
